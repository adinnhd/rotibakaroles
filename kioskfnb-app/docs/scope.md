# Kiosk FNB Scope

## Core MVP

- Customer can browse menu
- Customer can create order
- Customer can validate order before checkout
- Kitchen/admin can view incoming orders
- Kitchen/admin can update order status

## AI Features Later

- Computer vision to count people in front of kiosk camera
- Menu recommendation based on detected people count
- Voice order
- Voice order validation and editing before checkout

## Not Included Yet

- Payment gateway
- Inventory management
- Loyalty system
- Multi-branch SaaS
- Printer integration
